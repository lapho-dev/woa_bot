

## About

An automatic bot that uses open-cv recognision and pytesseract OCR, followed by adb connection to andriod emulator using adbutils library.


## Usage
0. clone the repo.
1. create python virtual env.
2. pip install requirements.txt
3. run android emulator (woa installed)
4. make sure is the android emulator is landscape + resolution 1920x1080
5. check the port no. and set it in src/config/settings.py. (default 16384)
6. run main.py
